# Desktop visualization of Red-Black tree
This is a desktop application with UI where you can *see how the RB tree works*.
## Functionality
* insert nodes
* delete nodes
* search nodes
* see the structure of the tree
## Technologies
* Java 8
* Java FX
* JDBC using mySQL
